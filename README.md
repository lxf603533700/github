# github介绍
github是一款分布式代码版本管理软件，相对于svn,能够较好的防止单点故障导致的代码版本历史销毁。

# github结构
工作区（写代码  add命令）->暂存区（添加但未提交 commit命令）->本地库（存储历史版本）
